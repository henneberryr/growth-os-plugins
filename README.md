# Growth OS Plugins for Claude Cowork

Official plugins from theCLICK (theclick.ai)

## Growth OS Quick Start v2.0.0

Guided setup that builds your AI workspace from your existing business assets in one session, plus an ongoing growth ideas engine.

### What it does

- Interviews you about your business, customers, and voice
- Creates persona, offers, and brand voice documents
- Generates growth ideas using the Customer Journey framework
- Builds your first marketing workflow
- Sets up task tracking

### Install

1. Download the `.zip` file from the Releases page.
2. Open Claude desktop app and switch to Cowork mode.
3. Install the plugin from the downloaded `.zip` file.
